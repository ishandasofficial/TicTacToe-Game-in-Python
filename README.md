# Tic Tac Toe 🎮

A simple Tic Tac Toe game built using Python and Tkinter.

## Features

- Two-player gameplay
- GUI built with Tkinter
- Win detection
- Draw detection
- Restart game option

## Requirements

- Python 3.any
- Tkinter

## Run the project

```bash
python main.py
```
